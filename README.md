# InboxAI

An AI-powered email client with semantic search, smart categorization, and multi-provider support.

## Tech Stack

- **Backend**: NestJS 10.x with Fastify, GraphQL, PostgreSQL + pgvector
- **Web**: Next.js 14 with React, TypeScript, Tailwind CSS
- **Mobile**: Flutter 3.x with Bloc state management

## Features

- Multi-provider support (Gmail, Outlook, IMAP)
- AI-powered semantic search
- Smart email categorization
- Email summarization
- Sender-based segregation
- Real-time sync
